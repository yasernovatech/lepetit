# ✅ چک لیست آپلود به Namecheap

## قبل از آپلود:

- [ ] فایل `.env.local` را حذف یا از آپلود خارج کنید
- [ ] پوشه `node_modules` را حذف کنید (نباید آپلود شود)
- [ ] پوشه `.git` را حذف کنید (نباید آپلود شود)
- [ ] بکآپ از فایلهای قبلی بگیرید

## بیلد پروژه:

- [ ] دستور `npm install` را اجرا کنید
- [ ] دستور `npm run build` را اجرا کنید
- [ ] بررسی کنید پوشه `out` ایجاد شده است
- [ ] محتویات پوشه `out` را بررسی کنید

## آپلود فایلها:

- [ ] به cPanel Namecheap وارد شوید
- [ ] File Manager را باز کنید
- [ ] به پوشه `public_html` بروید
- [ ] فایلهای قدیمی را حذف کنید (بکآپ گرفتهاید؟)
- [ ] تمام محتویات پوشه `out` را آپلود کنید
- [ ] فایل `.htaccess-namecheap` را به `.htaccess` تغییر نام دهید و آپلود کنید

## تنظیمات:

- [ ] فایل `.htaccess` را در `public_html` قرار دهید
- [ ] مجوزهای فایلها را بررسی کنید (644 برای فایلها، 755 برای پوشهها)
- [ ] SSL را فعال کنید (AutoSSL در cPanel)
