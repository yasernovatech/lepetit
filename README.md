# Le Petit - Next.js E-commerce

A modern e-commerce website built with Next.js 15, React 19, and Tailwind CSS.

## Tech Stack

- Next.js 15.5.6
- React 19.2.0
- TypeScript
- Tailwind CSS
- Radix UI Components

## Deployment

This project is configured for static export and can be deployed on Vercel.

### Build Command
```bash
next build
```

### Output Directory
```
out
```

### Environment Variables
No environment variables required for production.

## Development

```bash
npm install
npm run dev
```

Visit `http://localhost:3000`
