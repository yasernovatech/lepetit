# راهنمای آپلود سایت به هاست معمولی

## مرحله 1: ساخت فایلهای استاتیک

در ترمینال دستور زیر را اجرا کنید:

```bash
npm run build
```

این دستور پوشه `out` را با تمام فایلهای آماده برای آپلود میسازد.

## مرحله 2: آپلود به هاست

### روش 1: آپلود به روت دامنه (yoursite.com)

1. وارد cPanel شوید
2. به File Manager بروید
3. به پوشه `public_html` بروید
4. **تمام فایلهای داخل پوشه `out` را آپلود کنید** (نه خود پوشه out)
5. مطمئن شوید فایل `.htaccess` هم آپلود شده

### روش 2: آپلود به زیرپوشه (yoursite.com/shop)

اگر میخواهید سایت در زیرپوشه باشد:

1. ابتدا `next.config.mjs` را ویرایش کنید و این خطوط را اضافه کنید:
```javascript
basePath: '/shop',
assetPrefix: '/shop',
```

2. دوباره `npm run build` را اجرا کنید

3. محتویات پوشه `out` را در `public_html/shop` آپلود کنید

## مرحله 3: تنظیمات هاست

فایل `.htaccess` باید در روت سایت باشد. اگر وجود ندارد، این محتوا را در آن قرار دهید:

```apache
<IfModule mod_rewrite.c>
  RewriteEngine On
  RewriteBase /
  
  # Redirect to HTTPS
  RewriteCond %{HTTPS} off
  RewriteRule ^(.*)$ https://%{HTTP_HOST}%{REQUEST_URI} [L,R=301]
  
  # Handle Next.js routing
  RewriteCond %{REQUEST_FILENAME} !-f
  RewriteCond %{REQUEST_FILENAME} !-d
  RewriteRule ^(.+)$ /$1.html [L]
  
  # Handle trailing slashes
  RewriteCond %{REQUEST_FILENAME} !-f
  RewriteCond %{REQUEST_FILENAME} !-d
  RewriteRule ^(.+)/$ /$1.html [L]
</IfModule>

# Compression
<IfModule mod_deflate.c>
  AddOutputFilterByType DEFLATE text/html text/plain text/xml text/css text/javascript application/javascript application/json
</IfModule>

# Browser Caching
<IfModule mod_expires.c>
  ExpiresActive On
  ExpiresByType image/jpg "access plus 1 year"
  ExpiresByType image/jpeg "access plus 1 year"
  ExpiresByType image/gif "access plus 1 year"
  ExpiresByType image/png "access plus 1 year"
  ExpiresByType image/webp "access plus 1 year"
  ExpiresByType text/css "access plus 1 month"
  ExpiresByType application/javascript "access plus 1 month"
</IfModule>
```

## چک لیست نهایی

- [ ] دستور `npm run build` اجرا شد
- [ ] پوشه `out` ساخته شد
- [ ] محتویات پوشه `out` (نه خود پوشه) به `public_html` آپلود شد
- [ ] فایل `.htaccess` در روت قرار دارد
- [ ] سایت را در مرورگر تست کنید

## نکات مهم

1. **فقط محتویات پوشه out را آپلود کنید** نه کل پروژه
2. اگر سایت کار نمیکند، فایل `.htaccess` را چک کنید
3. Cache مرورگر را پاک کنید (Ctrl+F5)
4. اگر تصاویر نمایش داده نمیشوند، مسیرها را چک کنید

## حجم فایلها

پوشه `out` معمولاً بین 5-20 مگابایت است و آپلود آن 2-5 دقیقه طول میکشد.
