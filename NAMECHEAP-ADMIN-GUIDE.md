# 🎯 راهنمای استفاده از پنل مدیریت در Namecheap

## 📋 نحوه کار

چون Namecheap فقط Static Files را پشتیبانی میکند، پنل مدیریت به صورت زیر کار میکند:

1. شما تغییرات را در پنل اعمال میکنید
2. فایل JSON دانلود میشود
3. شما فایل را به هاست آپلود میکنید
4. تغییرات در سایت اعمال میشود

---

## 🔐 ورود به پنل

**آدرس:** `https://your-domain.com/admin`  
**رمز عبور:** `lepetit2025`

---

## 📝 مراحل ویرایش محتوا

### مرحله 1: ویرایش در پنل
1. به `your-domain.com/admin` بروید
2. رمز عبور را وارد کنید
3. تب مورد نظر را انتخاب کنید
4. تغییرات را اعمال کنید
5. روی "💾 ذخیره و دانلود" کلیک کنید

### مرحله 2: دانلود فایلها
بعد از کلیک روی ذخیره، 2 فایل دانلود میشود:
- `content.json` - فایل محتوا
- `راهنمای-آپلود.html` - راهنمای گام به گام

### مرحله 3: آپلود به هاست

#### الف) از طریق cPanel:
1. به cPanel هاست خود بروید
2. File Manager را باز کنید
3. به مسیر `public_html/data/` بروید
4. فایل `content.json` قدیمی را حذف کنید
5. فایل جدید را آپلود کنید

#### ب) از طریق FTP:
```
Host: ftp.your-domain.com
Username: your-username
Password: your-password
Port: 21

مسیر آپلود: /public_html/data/content.json
```

### مرحله 4: تست
1. سایت را باز کنید
2. Ctrl+F5 بزنید (رفرش کامل)
3. تغییرات را مشاهده کنید

---

## 🛍️ ویرایش محصولات

### ⚠️ نکته مهم
برای ویرایش محصولات باید سایت را دوباره بیلد کنید.

### مراحل:

#### 1. ویرایش در پنل
- تب "🛍️ محصولات" را باز کنید
- محصول مورد نظر را ویرایش کنید
- روی "💾 ذخیره همه محصولات" کلیک کنید

#### 2. دانلود فایل
فایل `data.ts` دانلود میشود

#### 3. جایگذاری فایل
```
data.ts → pro/components/kokonutui/data.ts
```

#### 4. بیلد مجدد
```bash
npm run build
```

#### 5. آپلود به هاست
پوشه `out/` را به `public_html/` آپلود کنید

---

## 🖼️ تغییر تصاویر

### مرحله 1: آپلود تصویر جدید
1. به cPanel → File Manager بروید
2. به `public_html/images/` بروید
3. تصویر جدید را آپلود کنید

### مرحله 2: ویرایش در پنل
1. به پنل مدیریت بروید
2. آدرس تصویر را تغییر دهید:
   ```
   /images/new-image.webp
   ```
3. ذخیره و آپلود کنید

## ⚡ نکات مهم

### ✅ انجام دهید:
- قبل از تغییر، از فایل قدیمی بکآپ بگیرید
- بعد از آپلود، سایت را تست کنید
- تصاویر را قبل از آپلود فشرده کنید

### ❌ انجام ندهید:
- فایل JSON را دستی ویرایش نکنید
- چند نفر همزمان ویرایش نکنند
- بدون بکآپ تغییر ندهید

## 🔄 فرکانس بروزرسانی

### محتوای متنی (سریع):
- زمان: 2-3 دقیقه
- مراحل: ویرایش → دانلود → آپلود

### محصولات (کند):
- زمان: 5-10 دقیقه
- مراحل: ویرایش → دانلود → بیلد → آپلود

---

## 🆘 رفع مشکلات

### مشکل: تغییرات نمایش داده نمیشود
**راهحل:**
1. کش مرورگر را پاک کنید (Ctrl+Shift+Delete)
2. Ctrl+F5 بزنید
3. مسیر فایل را چک کنید

### مشکل: فایل آپلود نمیشود
**راهحل:**
1. سایز فایل را چک کنید (باید کمتر از 2MB باشد)
2. دسترسی پوشه را چک کنید (755)
3. از FTP استفاده کنید

### مشکل: پنل باز نمیشود
**راهحل:**
1. آدرس را چک کنید: `your-domain.com/admin`
2. کش مرورگر را پاک کنید
3. از مرورگر دیگری امتحان کنید
