# راهنمای ثبت در Google Search Console

## مراحل ثبت:
1. به https://search.google.com/search-console بروید
2. "Add Property" را کلیک کنید
3. URL سایت را وارد کنید: https://lepetit.de
4. یکی از روشهای تایید را انتخاب کنید:
   - HTML file upload
   - HTML tag (در head سایت)
   - Google Analytics
   - Google Tag Manager

## بعد از تایید:
1. Sitemap را ثبت کنید: https://lepetit.de/sitemap.xml
2. URL Inspection برای صفحات مهم
3. بررسی Mobile Usability
4. بررسی Core Web Vitals
5. مانیتور کردن Performance

## فایلهای مورد نیاز:
- ✅ sitemap.xml (خودکار ایجاد میشود)
- ✅ robots.txt (ایجاد شده)

## نکات مهم:
- هر هفته گزارشها را بررسی کنید
- خطاهای Crawl را رفع کنید
- صفحات جدید را Index کنید
